Getting Started


If you are Getting started, there are a few required steps to follow, if not skip this part.

  1. Go to the Google Cloud Console
  2. Click on: Select a project
  3. Click on: Create a new project
  4. Name it: Youtube Playlists and click on CREATE
  5. On the left Navigation bar, click on Credentials
  6. Click on: CREATE CREDENTIALS
  7. Select: API Key. The creation of this credential should be instantaneous and be displayed in the list of credentials below.
  8. When this is done, click again on: CREATE CREDENTIALS and this time choose: OAuth client ID
  9. In the OAuth consent screen page: choose the External option and click on CREATE
  10. The OAuth consent screen page will continue, just name the application: Youtube Playlists and click on Save
  11. You will need to click on: CREATE CREDENTIALS -> OAuth client ID another time
  12. This time you will be redirected to the: Create OAuth client ID page. Choose Desktop app as the Application type and you should be done with the creation of credentials
  13. Download the file of OAuth 2.0 Clients IDs which should be name Desktop client 1

Upload credentials


You have the credentials you need to follow a channel in the Youtube Playlists follow-up application:

  • An API Key
  • An OAuth 2.0 Client ID
